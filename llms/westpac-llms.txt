# Westpac Banking Corporation

> Westpac Banking Corporation is Australia's oldest bank (founded 1817 as the Bank of New South Wales) and one of the "Big Four" ADIs. As a designated Consumer Data Right (CDR) data holder it exposes a public, unauthenticated Product Reference Data (PRD) API conforming to the DSB Consumer Data Standards. This is the only public API surface; consumer/account data sharing runs through the CDR Accredited Data Recipient consent model, not a public developer portal.

## APIs

- [Westpac CDR Product Reference Data API](https://www.westpac.com.au/about-westpac/innovation/open-banking/product-api/): Public, unauthenticated banking product catalogue (rates, fees, features, eligibility) under Australia's CDR. Base URL https://digital-api.westpac.com.au/cds-au/v1 ; send header x-v: 5. Two operations: Get Products (listProducts) and Get Product Detail (getProductDetail).
